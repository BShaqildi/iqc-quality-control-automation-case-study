# QC Process Flow

## Before iQC
1. User logs into ACES
2. Manually enters loan data
3. Performs QC review
4. Logs errors manually
5. Sends manual follow-ups
6. Repeats process across multiple loan touches

## After iQC
1. User opens iQC platform
2. Loan data auto-populates via API
3. Automated QC checks run
4. User reviews flagged issues
5. Errors are logged and assigned
6. Notifications are triggered automatically
7. Reporting updates in real time
